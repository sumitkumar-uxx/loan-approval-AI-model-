# loan-approval-AI-model-
A machine learning model that predicts whether a loan will be approved or not using a Random Forest Classifier. The dataset is preprocessed using Label Encoding and One-Hot Encoding. The model is trained on features like gender, income, education, employment status, property area, credit history, loan amount, etc.
